# LV100 アーケード過走演出パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

## 方針

LV60〜100の難しさ自体は大きく変えず、読み解ける範囲は守ったまま、テキストと演出で「世界が少し壊れていく」方向を足しました。

性的・際どい直接表現は入れていません。  
入れたのは、語尾の乱れ、エラーっぽい文、ゲームセンター側が止めに来るような空気、最後の暗転です。

## 主な変更

### 1. LV60以降のテキストに少しだけノイズを追加

例：

```text
世界が少しズレたにゃ
```

```text
店員さん、来るかも
```

```text
カウンタが、笑ってるにょ
```

ただし、ルール判定に必要な手の名前や合図は消さないようにしています。

### 2. 読み合いルールは維持

- `ためす` は必ず手の名前を出す
- `まよい` は表示中の手名が出る
- `ないしょ` はポーズ・手元を見る文を残す
- `あせり` は見えている手を示す

### 3. LV100到達で強制終了演出

100回あいこに到達すると、勝ち負けではなく、アーケード筐体側が限界になったような演出に変わります。

流れは、

```text
LV100
カウンタ上限です
```

```text
これ以上は
店員さんに怒られるにょ
```

```text
この先は
画面の外でね
```

```text
強制終了
称号を保存しました
```

のあと、暗転してタイトルへ戻ります。

### 4. タイトル画面に称号を表示

LV100完走後は、スタート画面の小さい文字が

```text
称号：LV100 完走者
```

に変わります。

## 注意

前回の「ためすを強制的にあいこへ補正する保険」は入れていません。  
今回も勝敗ロジックはそのままです。
