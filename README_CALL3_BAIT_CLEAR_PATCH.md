- assets/sounds/se_janken_call.mp3

## 1. じゃんけん効果音の使い方を変更

アップロードされた `se_janken_call.mp3` は、じゃんけん中の3拍目だけに使うようにしました。

- じゃん → 軽い電子音
- けん → 軽い電子音
- ぽん！ → se_janken_call.mp3

あいこの時も同じです。

- あいこ → 軽い電子音
- で → 軽い電子音
- しょ！ → se_janken_call.mp3

BGMとぶつかりにくくするため、1拍目・2拍目ではmp3を鳴らしません。

低負荷モードでは、mp3ではなく軽い電子音に戻します。

## 2. きもち：ためす のセリフを分かりやすく変更

以前の、

- ぐーに見せてみるね

は、意味が伝わりにくかったため変更しました。

新しい例：

- ぐーは ひっかけだよ
- ぐーに 見えても外して
- ぐーは 選ばないで

この場合は、ぐー以外を押すと、あいてが合わせてあいこになります。
