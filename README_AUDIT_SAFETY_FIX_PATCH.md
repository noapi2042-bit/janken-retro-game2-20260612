# 総合デバッグ修正パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

## 修正内容

調査で優先度が高かった不具合・不一致をまとめて修正しました。

### 1. 本番中のデバッグメニュー誤表示を防止

これまで「おと」ボタンを短時間に5回押すと、通常URLでもデバッグメニューが開ける状態でした。

今回から、デバッグメニューは `?debug=1` を付けた時だけ反応します。

### 2. メダル計算を BigInt 化

スコアアタック後半でメダルが大きくなっても、内部計算が壊れにくいようにしました。

- `pot`
- `あなた`
- `あいて`
- メダル最高記録

を BigInt ベースで扱います。

表示はこれまで通り、万・億・兆・京…の和風短縮表示です。

### 3. メダル表示の指数表記を廃止

`1e+04兆` のような表示にならないように、指数表記を使わないフォーマッタに変更しました。

### 4. ルール文言の統一

混乱しやすかった表示を少し統一しました。

- `ためす`：言った手以外
- `あせり`：先に見えた手
- `ないしょ`：手元の合図

### 5. スコアアタックタイマーの公平化

タイマーが説明文より先に進み始める問題を避けるため、ボタン有効化だけではタイマーを開始しないようにしました。

今は、説明やヒントが表示されたあと、少し待ってからカウント開始します。

また、ブラウザを一度隠した時はタイマーを止め、戻った後に再スタートします。

### 6. アルバム周りの英語表示を日本語化

- `LOCKED` → `未開放`
- `Gallery image` → `アルバム画像`
- 前後ボタンの aria-label も日本語化

### 7. モバイル操作抑制を少し弱める

全画面でブラウザ操作を完全に塞ぐ形を少し弱め、ボタン・メッセージ・手札周辺に限定しました。
