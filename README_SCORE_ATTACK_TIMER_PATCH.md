# スコアアタック制限時間パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

## 追加内容

TRUE END後のスコアアタックに、手を選ぶ制限時間を追加しました。

## 仕様

### 制限時間

LVが上がるほど、手を選べる時間が短くなります。

- LV 1〜15：10秒
- LV 16〜34：9秒
- LV 35〜54：8秒
- LV 55〜74：7秒
- LV 75〜89：6秒
- LV 90〜100：5秒

## 時間切れ

時間切れになると、その時点のメダルをあいてに取られてゲームオーバーになります。

ゲームオーバー時には、次のようなヒントが出ます。

- LV 42で時間切れ
- 8秒以内に選んでね

## UI

画面右上付近に小さく `LV / TIME` のカウントダウンを表示します。

残り3秒以下になると、赤く光って急かす演出に変わります。

## セリフ調整

世界観を壊しすぎないように、説明はキャラクターの言い方に寄せています。

例：

- 10秒以内に 手を選んでね
- 時間切れなら メダルをもらうね
- LVが上がるほど 待てる時間も短いよ
- 5秒だけ、待つね
