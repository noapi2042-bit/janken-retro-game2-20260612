# 初回「同じ手であいこ」裏切り修正パッチ

## 修正した問題

1回目の勝利後、2回目以降の練習で、

- まずは練習。
- 同じ手であいこ

というセリフが出ているのに、実際には相手が別の手を出して、あいこにならない場合がありました。

## 原因

このセリフは見た目上は「同じ手であいこ」と言っていましたが、内部的には `psychEvent` が設定されておらず、普通のランダムじゃんけんとして処理される可能性が残っていました。

つまり、表示はチュートリアルなのに、判定は普通じゃんけんになっていました。

## 対応

このセリフを出す時に、内部的にも

- きもち：みてる
- あなたに合わせる
- プレイヤーが出した手に、相手が合わせる

という特殊イベントとして登録するようにしました。

## 修正後の挙動

「まずは練習。同じ手であいこ」が出ている時は、

- あなた：ぐー → あいて：ぐー
- あなた：ちょき → あいて：ちょき
- あなた：ぱー → あいて：ぱー

になります。
