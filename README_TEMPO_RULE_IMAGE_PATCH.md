# テンポ加速・3回目サポート非表示・画像表示改善パッチ

## 差し替えるファイル

リポジトリ直下の以下3ファイルを差し替えてください。

- index.html
- style.css
- script.js

assets/images と assets/sounds はそのままで大丈夫です。

## 主な修正

### 1. じゃんけんの掛け声テンポを段階的に加速

1試合内であいこが続くほど、じゃんけんの掛け声を少しずつ速くします。

目安：

- 0〜2あいこ：ゆっくり
- 3〜6あいこ：少し速い
- 7〜14あいこ：テンポよく
- 15〜29あいこ：速め
- 30あいこ以降：かなりテンポよく

セリフの文字送りは前回の「読める速度」を維持し、じゃんけんポン部分だけテンポを上げています。

### 2. 2回目はサポートあり、3回目以降はサポートなし

普通勝利後の2回目では、

きもち：あわせたい
同じ手

のように、下段サポートを表示します。

3回目以降は、

きもち：あわせたい

だけにして、少し難易度が上がるようにしました。

### 3. キャラ画像表示を改善

画像が出ない原因候補として、JS側の事前読み込み処理で画像表示まで進まないケースがあり得るため、キャラ画像はDOM上の img に直接読み込ませる方式に変更しました。

画像候補はこれまで通り、

- その表情の画像
- character_draw_01.png
- character_normal_01.png
- character_happy_01.png
- character_worried_01.png
- character_excited_01.png

の順で探します。

### 4. 画像ローダーの lazy 指定を解除

JS内で作る画像ローダーは、スマホでも eager 読み込みにしました。
これで、軽量モード時に画像が遅延しすぎて仮顔のままになる可能性を減らしています。
