# あせり演出変更・TRUE END後100回完走パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

assets/images と assets/sounds はそのままで大丈夫です。

## 主な修正

### 1. きもち：あせり のセリフから手の名前を削除

`きもち：あせり` では、セリフ内に「ぐー」「ちょき」「ぱー」と書かないようにしました。

代わりに、あいて側の手札欄に先に手が表示されます。

例：

- セリフ：あっ… もう出しちゃった
- あいて側の手札：すでに表示済み

プレイヤーは、セリフではなく、あいての手札欄を見て同じ手を出す形です。

### 2. あせり時の手札に注意表示

あいての手札欄に `先に出てる` の小さな表示と光る演出を追加しました。

### 3. TRUE END後のスコアアタックに100回完走を追加

TRUE END後、あいこが100回まで続いたら「完走」として強制終了します。

完走時は、

- BGMを切り替える
- 手札を一度「？」に戻す
- 最後は見えない一手として、勝ち負けを強制的に決める
- その後タイトルへ戻る

という流れになります。
