# README_HIDE_POSE_TUTORIAL_PATCH

このパッチでは、**「隠した手」パラメーター**を段階的に学べるように調整しています。

## 変更点
- 初回の「隠した手」は、キャラクター横の**手アイコン**で分かりやすく表示
- 2回目は**アイコン + キャラクターポーズ**で半チュートリアル化
- 3回目以降は**キャラクターの手ポーズだけ**で読ませる構成に変更
- 隠した手用の立ち絵は、**手が顔の近くにある比較的安全なポーズ**を優先採用
- セリフは手の名前を直接言わず、
  - 初回: 「この手をおぼえてね」
  - 2回目: 「つぎはこの手かな」
  - 3回目以降: 「見たままで合わせてね」
  など、画像誘導向けの文に調整

## 主な対象ファイル
- index.html
- style.css
- script.js

## 実装メモ
- `state.feelingCueSeen.hide` 相当で、同一プレイ中の「隠した手」出現回数を管理
- `characterPoseHint` 要素を追加し、初回/2回目のみ手アイコンを表示
- `HIDE_TUTORIAL_POSE_LIBRARY` で安全寄りの手ポーズを指定

## 反映方法
既存の
- `index.html`
- `style.css`
- `script.js`

を、このパッチ同梱版に置き換えてください。
