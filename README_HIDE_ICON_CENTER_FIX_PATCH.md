# 内緒アイコン中央寄せ修正パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

## 修正内容

`ないしょ` パラメータ時にキャラクター右上へ出る手アイコンについて、
特に `ちょき` が枠の右上へ寄って見える問題を見た目ベースで補正しました。

### 調整内容

- アイコン枠の中身を `flex` で中央寄せに統一
- 画像を `display:block` + `margin:auto` にして中央配置を安定化
- `object-position:center center` を追加
- `ちょき` だけ視覚的重心を少し左下へ寄せる微調整
- `ぐー` / `ぱー` も軽く中央寄せを安定

## 備考

- ロジック変更はありません
- 表示位置のみの微調整です
- ベースは `janken_lv100_text_tune_patch` です
