# コナミ風デバッグ復活パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

## 追加内容

通常URLでも、隠しコマンドでデバッグモードを開けるようにしました。

`?debug=1` を付けなくても、検証時に「こたえ表示」とデバッグパネルを使えます。

## 開き方

### キーボード

以下を順番に入力します。

```text
↑ ↑ ↓ ↓ ← → ← → B A
```

### スマホ / タッチ操作

タイトル画面で、筐体画面のだいたいの位置を順番にタップします。

```text
上 上 下 下 左 右 左 右 中央 中央
```

成功すると、

```text
ひみつのテスト
こたえ表示ON
```

と出て、デバッグパネルと左下のこたえ表示が開きます。

## 使いどころ

- 負けた時に、実際の読みが合っていたか確認
- `ためす` の「言った手以外」が正しいか確認
- `まよい` の表示中の手が合っているか確認
- スクリーンショットで報告しやすくする

## 注意

- 通常の5回連打では開かないままです。
- 誤爆しにくいように、スマホ版はタイトル画面でだけ反応します。
- 閉じたい時はデバッグパネルの `×`、またはキーボードの Esc で閉じられます。
