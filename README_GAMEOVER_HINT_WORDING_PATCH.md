# ゲームオーバー誘導セリフ調整パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

## 修正内容

アルバム回収の誘導セリフにあった、

```text
負けた時の私も、
少しだけ見てほしいかも。
```

という表現を修正しました。

この文だと「私の負けた姿を見てほしい」という意味に読めてしまう可能性があるため、現在のゲーム内容に合わせて、次の表現に変更しています。

```text
まだ見てない終わり方も、
少しだけ見てほしいかも。
```

## 意図

- 「プレイヤーが負けて終わるルート」への誘導だと分かる
- キャラクター自身の負け姿を指しているように見えにくい
- 世界観を壊さず、やわらかい言い方にする
