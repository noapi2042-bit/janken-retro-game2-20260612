# あいこコミュニケーション強化パッチ

このパッチは、「安心感」と「ゲーム性」の間を狙って、あいこを続けるための心理イベントを整理したものです。

## 差し替えるファイル

リポジトリ直下の以下3ファイルを差し替えてください。

- index.html
- style.css
- script.js

assets/images と assets/sounds はそのままで大丈夫です。

## 主な修正

### 1. 「ためす」を運ゲーにしない

以前の「ためす」は、見せた手と本当の手がズレていて、場合によっては2択運になりやすい構造でした。

今回からは、

- きもち：ためす
- 例：ぱーっぽく見えるかな？

の場合、

- プレイヤーが「ぱー」を選ぶ → ひっかかった扱い
- プレイヤーが「ぐー」または「ちょき」を選ぶ → 相手がその手に合わせて、あいこ

になります。

つまり、「見せ手を外せたら成功」というルールです。

### 2. 「みてる」を追加

新しい気持ちとして、

- きもち：みてる
- ルール：あなたに合わせる

を追加しました。

これは、プレイヤーが先に手を出すと、相手が後から同じ手を出してあいこにしてくれるイベントです。

セリフ例：

- 先に出してみて
- 今度はわたしが見るね
- あなたの手に合わせるね

### 3. 使いすぎないバランス

「みてる」は何を出してもあいこになるため、出しすぎるとゲーム性が薄くなります。
そのため、チュートリアル・練習区間・スコアアタック中に、息継ぎや安心感としてたまに出る程度にしています。

### 4. デバッグHUDも更新

デバッグHUDで以下が分かるようにしました。

- ためす：見せ手以外ならあいこ
- みてる：どの手でも相手が合わせる
- 実際に選んだ後の、あなたの手・あいての手
