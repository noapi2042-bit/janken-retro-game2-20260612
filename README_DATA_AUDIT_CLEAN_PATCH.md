# データ調整・不具合点検パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

画像・音声ファイルの追加はありません。

## 修正内容

### 1. 重要：隠した手ポーズ用データの初期化順を修正

`HIDE_TUTORIAL_POSE_LIBRARY` が `CHARACTER_POSE_META` より前で作られており、ブラウザ実行時に初期化順の問題が起きる可能性がありました。

今回、`CHARACTER_POSE_META` 定義後に移動しました。

### 2. ためすの文言を単純化

`きもち：ためす` は、二重否定を避けて次のルールに統一しました。

- セリフで言った手は、ひっかけ
- その手以外を出すと、あいてが合わせてあいこ

例：

- セリフ：ぱーにしようかな
- 正解：ぱー以外

以下のような混乱しやすい言い方は使わない方針です。

- ぱーは選ばないで
- ぱーに見えても外して
- 言葉の逆

### 3. まよい文字流しを復元

前回入れた、`きもち：まよい` の文字が切り替わる時に流れて見える演出を復元しました。

### 4. 長時間待機後の掛け声安定化を復元

20〜30秒ほど待ってから押した時に、スマホで効果音や掛け声がズレて感じる可能性への対策を復元しました。
