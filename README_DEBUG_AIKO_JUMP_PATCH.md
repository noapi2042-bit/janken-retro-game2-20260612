# デバッグあいこ回数ボタン変更パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

## 修正内容

デバッグモード内の `あいこ 5 / あいこ 10 / あいこ 15` を、現在の確認作業向けに変更しました。

## 新しいボタン

### あいこ +5

現在のあいこ回数に `+5` します。

例：

```text
あいこ 10回の時に押す → あいこ 15回
```

### あいこ +10

現在のあいこ回数に `+10` します。

例：

```text
あいこ 70回の時に押す → あいこ 80回
```

### あいこ 90へ

あいこ回数を `90回` にジャンプします。

LV100手前の確認用です。

TRUE END後のスコアアタック中に押すと、通常のチャンス/ファイナル状態にはせず、スコアアタック用の状態として扱います。

## 上限

最大は `100回` で止まるようにしています。  
`+10` を押して100を超える場合でも、100回までです。
