# きもち別セリフ整理パッチ

## 修正方針

読み合いの文言を、きもちごとに整理しました。

| きもち | セリフの役割 | プレイヤーの判断 |
|---|---|---|
| すなお | 本当の手を言う | 表示された手を出す |
| ためす | 言った手がワナ | 表示された手以外を出す |
| あわせたい | その手を出してほしい | 表示された手を出す |
| ぴったり | 同じ手で合わせる | 表示された手を出す |
| まよい | 今見えている手が合図 | 押す直前の手を出す |
| ないしょ | 手元・ポーズが合図 | 見えた手を出す |
| あせり | 先に見えた手が合図 | 見えた手を出す |
| みてる | あなたが先に出す | どれでもOK |

## すなお

今後は、`ためす` の「出すね」と混ざらないように、こういう文に寄せました。

```text
ほんとは
ぐーだよ
```

```text
ぐーで
そのまま
```

## ためす

前回の方針を維持しています。

```text
ぐーを
出すね
```

この場合、ぐーはワナです。

```text
ぐーを押す → 負け
ちょき / ぱーを押す → あいこ
```

## あわせたい

プレイヤーに出してほしい文にしました。

```text
ぐーを
出してね
```

```text
ぐーで
合わせてね
```

```text
ぐーを
出してくれる？
```

この場合は、ぐーを出すとあいこです。

## ぴったり

TRUE END後のぴったり系は、合わせる文にしました。

```text
ぐーで
ぴったり
```

```text
ぐーで
合わせてね
```

## 維持したもの

- 勝敗ロジック
- ためすの「言った手以外ならあいこ」
- デバッグの `あいこ +5 / +10 / 90へ`
- スコアアタック
