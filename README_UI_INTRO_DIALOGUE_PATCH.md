# UI重なり修正・初回説明ゆっくり化・会話ヒント改善パッチ

## 差し替えるファイル

リポジトリ直下の以下3ファイルを差し替えてください。

- index.html
- style.css
- script.js

assets/images と assets/sounds はそのままで大丈夫です。

## 主な修正

### 1. スマホ画面の重なりを調整

スクリーンショットで見えていた、

- 進行チップがセリフ枠に近すぎる
- 文字や枠が重なって見える
- セリフ欄の上側が詰まって見える

という点を優先して、最終CSSで間隔を固定しました。

特に、

- 進行チップの下に余白を追加
- セリフ枠の開始位置を調整
- セリフ文字のサイズを少し抑える
- キャラ表示を少しだけ下げて重なりにくくする

を行っています。

### 2. 最初の説明をゆっくり表示

スタート直後の説明を2段階に分けて、文字送りを遅くしました。

- まずは 30まい。メダルをあつめよう
- あいこが続くと、メダルも大きくなるよ

一番最初は丁寧に読ませ、その後のじゃんけんテンポは今まで通り、あいこが続くほど速くなります。

### 3. 勝利後の会話を自然なヒントに修正

勝利後のセリフを、直接的なチュートリアル文ではなく、キャラクターの感情を混ぜたヒント文に変更しました。

例：

- でもね、あいこだと もう少し一緒にいられるの。
- 私の「きもち」も、少しだけ見てみて。
- あわせたい時は、同じ手で待ってるね。
- 試してる時は、見せた手にひっかからないで。
