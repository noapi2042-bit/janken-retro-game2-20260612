# 読みやすさ・注意SE・UI密度ブラッシュアップパッチ

このパッチは、セリフが速すぎる問題、注意が必要な読みの演出不足、スコア/あいこ条件表示の小ささ、手札表示と下ボタンの間延び感を調整したものです。

## 差し替えるファイル

リポジトリ直下の以下3ファイルを差し替えてください。

- index.html
- style.css
- script.js

assets/images と assets/sounds はそのままで大丈夫です。

## 主な修正

### 1. セリフ表示をゆっくりに変更

文字送りを少し遅くして、読んで考えるリズムを作りました。

- 通常セリフ: 少しゆっくり
- 読み合いヒント: さらに少しゆっくり
- チャンス中の自動セリフ: 切り替え間隔を 2.6秒 → 3.6秒
- TRUE END / シーン会話: 文字送りを少しゆっくり

### 2. 注意が必要な読みへ効果音を追加

以下の読みが出た時だけ、短い注意音を入れます。

- ためす
- ないしょ
- まよい
- あせり

普通の「すなお」「あわせたい」では鳴らしすぎないようにしています。

### 3. あいこ条件表示を大きくしました

「チャンスまであと○」「さいごまであと○」が小さかったので、少し大きく見えるナビ表示にしました。

### 4. スコア表示を4項目向けに整理

現在は「あなた / メダル / あいて / あいこ」の4項目なので、4列表示に合わせて整理しました。

### 5. 手札表示とボタンの間延びを調整

あなた・VS・あいての表示を少し中央に寄せ、下のぐー・ちょき・ぱーボタンも少し密度を上げました。

### 6. キャラ画像の表示補助

軽量モードでも最初のキャラ画像だけは先読みするようにしました。
表情画像が見つからない場合は、以下の共通画像を探します。

- character_draw_01.png
- character_normal_01.png
- character_happy_01.png
- character_worried_01.png
- character_excited_01.png
