# 演出・立ち絵整合性ブラッシュアップパッチ

## 差し替えファイル
- index.html
- script.js
- style.css

assets/images と assets/sounds はそのままで大丈夫です。

## 主な修正

### 1. 立ち絵とセリフの整合性を強化
- 読み合いイベント時に、
  **実際に見せたい手・隠したい手・ニュートラルな手** を考慮して立ち絵候補を優先選択するようにしました。
- 例:
  - `あわせたい / 同じ手` は、実際の手に合うポーズを優先
  - `ためす` は、見せかけの手に近いポーズを優先
  - `ないしょ` は、目立つ手サインを避け、控えめな立ち絵を優先

### 2. 顔寄りの演出を追加
- 立ち絵ごとに **寄り / 少し斜め / 左右寄せ** をランダムに切り替え、
  いつも同じ見え方にならないようにしました。
- 顔をやや大きめに見せる方向へ寄せています。

### 3. セリフの言い回しを微修正
- 画像の見せ方とズレにくい短文へ調整しました。
- とくに `ためす / ないしょ / まよい` は、
  **「まだ続けたい」気持ちが伝わりやすい** 言い回しに寄せています。
