# UI重なり除去パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

assets/images と assets/sounds はそのままで大丈夫です。

## 修正内容

### 1. 進行チップを一旦非表示

実機スクリーンショットで、

- ふつう
- あいこ0
- 目標30

の3連チップが、セリフ枠と重なり続けていました。

このチップはレイアウトを複雑にしていたため、通常画面では一旦非表示にしました。

### 2. セリフ枠を独立

セリフ枠を上部に独立して置き、チップや他のUIと重ならないようにしました。

### 3. キャラ・手札・ボタンの収まりを再調整

チップを消したぶん、キャラと手札の表示を自然に詰め直しています。
