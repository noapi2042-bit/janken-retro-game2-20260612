# テンプレ読み・難易度調整パッチ

このパッチは、セリフの言い回しが多すぎて疲れやすい問題を修正したものです。

## 差し替えるファイル

リポジトリ直下の以下3ファイルを差し替えてください。

- index.html
- style.css
- script.js

assets/images と assets/sounds はそのままで大丈夫です。

## 修正内容

### 1. セリフをテンプレ化しました

各きもちごとのセリフを、基本1〜2パターンに絞りました。

- すなお：つぎは ○○だよ
- あわせたい：○○で あいこにしよ
- さそい：○○が よさそうだよ
- まよい：○○かな…やっぱ△△
- かくす：かくしてるのは ○○
- あせり：あっ…○○って言った

### 2. 難易度を段階化しました

序盤は簡単にして、終盤だけ難しくしました。

- 序盤：すなお / あわせたい が中心
- 入り口：まよい が少し出る
- 中盤：さそい が少し増える
- 終盤：さそい / まよい / あせり / かくす が混ざる

### 3. ベリーハードになりすぎないように調整

「かくす」は終盤だけ低確率にしました。
普段は、少し考えれば分かる型を中心にしています。

### 4. きもちの読み方を短くしました

- きもち：すなお / そのまま
- きもち：あわせたい / 同じ手
- きもち：さそい / おすすめはワナ
- きもち：まよい / あとが本音
- きもち：かくす / かくした手
- きもち：あせり / そのまま
