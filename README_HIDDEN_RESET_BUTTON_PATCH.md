# 隠し初期化ボタン追加パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

## 追加内容

通常公開ではデバッグメニューを開けないまま、タイトル画面に小さな `初期化` ボタンを追加しました。

## 使い方

タイトル画面の左上付近に、薄く `初期化` が出ます。

誤操作防止のため、2回押しです。

1. `初期化` を押す
2. 表示が `もう一度` に変わる
3. 3秒以内にもう一度押す
4. セーブデータを消して最初から開始

## 消えるデータ

`janken` で始まる localStorage の保存データを削除します。

つまり、

- アルバム解放
- TRUE END後の記録
- メダル記録
- チュートリアル進行

などを初期化できます。

## 表示場所

通常プレイ中には出さず、タイトル画面だけに表示します。
