# LV100 テキスト微調整パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

## 方針

前回の方向性を保ちつつ、語尾を `にょ` 固定にせず、もう少し昭和・方言・変な筐体ノリに寄せました。

直接的な下ネタや際どい表現は入れていません。  
匂わせは、最後の暗転と「画面の外でね」程度に抑えています。

## 主な調整

### 1. LV60〜100の語尾を分散

使う語尾を少し増やしました。

- にゃ
- だべ
- だどん
- だわさ
- なのだ
- じゃよ

ただし、読み解きに必要な `ぐー / ちょき / ぱー` は消さないようにしています。

### 2. LV80〜100のセリフを少し不安定に

例：

```text
カウンタが、笑ってるだどん
```

```text
字幕が昭和になってきたじゃよ
```

```text
筐体の音が変だわさ
```

ゲーム性を難しくするのではなく、世界観が壊れていく感じだけを足しています。

### 3. ルール情報は維持

- `ためす` は必ず手の名前を出す
- `まよい` は表示中の手名が出る
- `ないしょ` は手元・ポーズを見る文を出す
- `あせり` は見えている手名が出る

### 4. LV100到達演出

100回あいこに到達すると、

```text
LV100
カウンタ上限です
```

```text
これ以上は
店員さんに怒られるだべ
```

```text
この先は
画面の外でね
```

```text
強制終了
称号を保存しました
```

の流れで暗転し、タイトルへ戻ります。

### 5. タイトル称号

LV100完走後、スタート画面に

```text
称号：LV100 完走者
```

が残ります。

## 注意

- 勝敗ロジックは変更していません。
- 前に却下された「ためすを必ずあいこに補正する保険」は入れていません。
- 画像追加・クリアイラスト追加はありません。
