# サウンドボタン5回デバッグ復活パッチ

## 差し替えるファイル

- index.html
- style.css
- script.js

## 追加・修正内容

スマホでデバッグモードを出しやすくしました。

## 一番かんたんな出し方

ゲーム画面右上の `おと ON / おと OFF` ボタンを、短い間に5回押します。

```text
おとボタンを 5回 連続タップ
```

成功すると、

```text
ひみつのテスト
こたえ表示ON
```

と出て、デバッグパネルと左下のこたえ表示が開きます。

## 以前の隠しコマンドも継続

キーボード：

```text
↑ ↑ ↓ ↓ ← → ← → B A
```

スマホの画面タップ：

```text
上 上 下 下 左 右 左 右 中央 中央
```

も残しています。

## スマホ隠しコマンドの修正

前の版では、中央タップがスタートボタンに当たってしまい、途中でゲームが始まることがありました。

今回から、隠しコマンド入力が始まった後のタップはゲーム操作に流さないようにしました。

## 注意

サウンドボタン5回押しは、通常公開URLでも使えます。`?debug=1` は不要です。
